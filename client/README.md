## Deployment

`master` branch is automaticaly deployed by Netlify.

To get a local build, run `npm run build`, ensure that you have a `.env` file
with at least:
```
DBSS_GAME_SERVER_HOST=http://xxx.xxx.xxx.xxx:xxxx
```
